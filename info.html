Introduction

Traditou intercepts the original subtitle files sent from the website servers, parses them and displays the original subtitles along with English subtitles on the bottom, with Google translate.

Why Traditou?

This extension can be used with the trifecta of French Canadian streaming websites: Tou.tv, Noovo.ca, and TeleQuebec, with the goal of French language learning in mind.

Traditou not only matches words closely, it also transforms TV-style rolling captions (which are visually difficult to follow) into more readable streaming service style subtitles.

Tutorial

First step to displaying auto-translated English subtitles:
Right click to select "Translate to English" or use Chrome's built-in Google translate icon on the top right to translate the page to English.
Note: for Telequebec, to translate the page, right click on the bottom video progress bar.

How to display both French and English subtitles:

Make sure Traditou's French + English subtitles mode is enabled.

Display only English subtitles

Toggle Traditou's subtitle mode to English subtitles, and also still leave Google's auto-translation enabled.

Display only French subtitles

Toggle Traditou's subtitle mode to French subtitles. Now it doesn't matter if Google's auto-translation is enabled or not.

Change subtitles' appearance

The styling of the subtitle boxes for iOS systems are in settings, under accessibility. You can select an existing style for your subtitles or create a customized style. Once you have made your choice, be sure to restart your browser to see the changes.

FAQs

  1. Subtitles are not showing up. What's wrong?

    Sometimes Google translate is slow to react. Try translating the page to English after the video is loaded on screen. Or try toggling the Google translate option from translating to not translating, and then back to translating.

    Also, double check if the video originally comes with subtitles, if not, no translated subtitles can be provided.

  2. The show that I want to watch is not on Tou.tv, Noovo, or TeleQuebec, will you support the website that I watch too?

    If you are looking for support for Netflix or Amazon Prime, there are a few Chrome extensions that already do that. If the website that you use is not any of the above, you can send me an email so I can add support for the website that you watch.

  3. Having trouble accessing the video contents because you are outside of Canada?

    You can use a VPN to change your IP address in order to access Canadian content.

  4. Can I use Traditou to translate to a language that's not English?

    Yes. As long as Google Translate supports that language.

  5. The subtitles seem to flicker a lot. What should I do?

    You can try to see if restarting your browser helps.
